# PyAxo Commands

This document lists the current commands available in PyAxo and the commands planned for future versions.

PyAxo is still in early beta development, so most features are being built step by step.

## Current Commands

These commands are already implemented in the current version.

### `!ping`

Checks if PyAxo is online.

Example:

`!ping`

Expected response:

`🟢 Status: ONLINE`

---

### `!regras`

Shows the rules of the Python Academy BR Discord server.

Example:

`!regras`

Expected response:

The bot sends the server rules.

## Planned Commands

These commands are planned but not implemented yet.

### `!ajuda`

Will show the list of available commands and explain how to use them.

Status: planned

---

### `!sobre`

Will explain what PyAxo is and what Python Academy BR is.

Status: planned

---

### `!aulas`

Will list the available lessons in the Python Academy BR course.

Status: planned

---

### `!aula`

Will show the content of a specific lesson.

Example:

`!aula 0`

Status: planned

---

### `!atividade`

Will show the activity for a specific lesson.

Example:

`!atividade 0`

Status: planned

---

### `!perguntar`

Will allow students to ask basic Python questions using a keyword-based answer system.

Example:

`!perguntar o que é uma variável?`

Status: planned

---

### `!perfil`

Will show a student's profile, including points, completed lessons, and level.

Status: planned

---

### `!pontos`

Will show a student's current points.

Status: planned

---

### `!rank`

Will show the student ranking based on points.

Status: planned

## Planned Staff Commands

These commands are planned for instructors, monitors, and administrators.

### `!addpontos`

Will manually add points to a student.

Status: planned

---

### `!removerpontos`

Will manually remove points from a student.

Status: planned

---

### `!aprovar`

Will manually approve a student's activity.

Status: planned

---

### `!setnivel`

Will manually change a student's level.

Status: planned

## Notes

Only the commands listed under **Current Commands** are currently implemented.

Commands listed under **Planned Commands** are part of the project roadmap and will be developed in future versions.
