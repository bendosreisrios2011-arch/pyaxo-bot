# PyAxo Bot

PyAxo is a Discord bot I am building for **Python Academy BR**, a free Python course hosted on Discord.

The bot is designed to help organize the course and support beginners with commands, lessons, rules, and future progress tracking.

## Python Academy BR

Python Academy BR is a free Python course created in Brazil inspired by structured computer science learning models.

The course is made for people who want to learn Python, especially those who may not be able to pay for an expensive programming course.

## Course Structure

The course is currently planned with the following lessons:

* Lesson 0 — Introduction to Python
* Lesson 1 — `print()`, `input()`, and variables
* Lesson 2 — Data types
* Lesson 3 — Conditionals
* Lesson 4 — Mathematical operators
* Lesson 5 — Loops
* Lesson 6 — Lists
* Lesson 7 — Functions
* Lesson 8 — Final project

## Why I Started This Project

I started this project after noticing that, despite the rapid evolution of technology and artificial intelligence in recent years, many people still remain disconnected from these developments because of misinformation, lack of motivation to learn, or financial limitations.

With Python Academy BR, I want to make programming knowledge more accessible. Through this free course, I aim to share what I am learning and build tools that can help other beginners start their journey in Python and computer science.
